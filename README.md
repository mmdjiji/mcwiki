# Minecraft中文文档 - 关于

Minecraft Wiki 是对 [wiki.vg](https://wiki.vg) 的全站中文翻译，是 Minecraft 开发与二次开发的详细中文文档，其中包含整套C/S通讯协议及各类API，及对各种问题的解答。

本文档由吉吉创始并完成前期翻译，目前也正在翻译中，欢迎各位热爱 Minecraft 的开发者一起加入进来。
